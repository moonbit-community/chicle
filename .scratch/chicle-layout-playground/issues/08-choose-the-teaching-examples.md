# Choose the teaching examples

Type: grilling
Status: resolved
Blocked by: 02, 06

## Question

Which small sequence of editable Layout Documents should teach Chicle's most important layout capabilities, and what annotations or labels are necessary for each example without turning the page into full API documentation?

## Answer

The page ships three compact documents: a nested Flex dashboard, an explicitly placed Grid workspace, and a relative hero with absolute overlays. Node labels describe visual intent but remain decorative.
